Drupal 8 Deeper Dive

Drupal 8 is coming. Amazee Labs is already building customer sites with Drupal 8. In this presentation I will show you which new and changes functionalities we liked the most or which ones forced us to change our workflows. It's targeted to Editors, Sitebuilders, Frontend and Backend. So everybody will learn what is new in D8.
